## 📝 API文档

### 核心接口
```
POST /api/v1/auth/login           # 用户登录
POST /api/v1/auth/register        # 用户注册
GET  /api/v1/projects             # 获取项目列表
POST /api/v1/recording/start      # 开始录制
POST /api/v1/recording/stop       # 停止录制
POST /api/v1/test-cases/:id/execute # 执行测试用例
```
